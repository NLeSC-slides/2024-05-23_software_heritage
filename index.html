Save Your Code++

Make sure your repository contains the following:

  • AUTHORS
    A list of all the persons that need to be credited for the software.
  • codemeta.json
    A linked data metadata file will assist in indexing your source code in the Software Heritage archive and provide an easy way to link to other related research outputs. See the CodeMeta initiative for more information and the CodeMeta generator tool.
  • LICENSE
  • README
